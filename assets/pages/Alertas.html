Juntando uma String

Se um dos lados for uma string por exemplo: ("30"), o javascript não soma, ele junta

Exemplo 1: Seguindo o exemplo de e = "30" + 2 = "302"

Exemplo 2: Quando você colocar var f = number(a) e a variavel for var a = "20". Você nao precisa escrever variavel, só escrever number(a).

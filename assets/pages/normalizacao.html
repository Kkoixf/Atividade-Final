O que é normalização?

O que é normalização no CSS? É um conjunto de estilos CSS que “zeram” ou ajustam as diferenças padrão dos navegadores para os elementos HTML. Por que é importante? Cada navegador (Chrome, Firefox, Edge, Safari) tem seus estilos padrões próprios para tags, tipo margens, fontes, espaçamentos, etc. Isso pode fazer seu site parecer diferente dependendo do navegador. Como funciona? A normalização padroniza esses estilos básicos para que a aparência inicial seja igual em todos os navegadores. Diferente do reset CSS, que geralmente zera quase tudo, a normalização mantém estilos úteis, mas corrige as diferenças.
